## MakeShift3d.com Express App
API for makeshift3d.com.

### NPM Commands ###

**install dependencies**

`npm install`

**start local dev server** [default: localhost:3000]

`npm run start`

**run tests**

`npm run test`

**watch tests**

`npm run test:watch`

**manual deploy** (requires ssh cert)

`npm run deploy`

**migration commands**

`migrate:create`

`migrate:up`

`migrate:down`

`migrate:redo`
